# Corona Stats

A statistics app to provide details about Corona aoutbreak in india

[***Link to Releases page***](https://gitlab.com/AuroraOSS/CoronaStats/-/releases)

# Features

* Free / Libre software
* Daily Report
* State wise reports
* Graphical Stats
